ROS Serial cho STM32

Nguyen Nhan Tinh
Nov 21, 2020 2 min read

Giới thiệu:

ROS chỉ được hỗ trợ chính thức trên hệ điều hành nhân linux. Vì vậy khi thực hiện những tác vụ đòi hỏi thời gian thực, độ chính xác về timing cao thì hệ thống linux không đáp ứng được. Bài viết hướng dẫn đưa ROS serial lên STM32 để có thể đáp ứng được yêu cầu.

Phần cứng trong bài viết được sử dụng là board Nucleo STM32F746ZG chạy trên FreeRTOS. Có thể sử dụng board khác bằng cách sử dụng STMcubeIDE để generate lại board phù hợp và thay đổi config GPIO. Ngoài ra còn 1 board để chạy ROS core là RaspberryPi 4. 2 board giao tiếp qua USB CDC.

Nội dung:

Ta sẽ cài ROS lên một thiết bị chạy linux, ở đây tôi dùng RaspberyPi4. Các bạn có thể tham khảo việc cài đặt ROS lên PI ở đây:

Download source code cho STM ở đây: https://github.com/tinhnn/rosserial_stm32f7xx

Sử dụng STM32CubeIde để build và flash. Source code được viết cho STM32F7 series, Nếu sử dụng board khác thì mở file ROSSerial_STM32.ioc config sang board mà bạn sử dụng rồi gen lại code.

Kết nối Pi với STM32 qua USB.

Trên Pi, mở Terminal 1 và chạy ROS core:

$ roscore

Mở Terminal 2 và chạy command:

$ rosrun rosserial_python serial_node.py _port:=/dev/ttyACM0 _baud:=115200

Mở Terminal 3 và chạy command:

$ rostopic list
/chatter
/diagnostics
/rosout
/rosout_agg
$ rostopic echo /chatter
data: "Welcome to ROS Serial for STM32f7 series"
